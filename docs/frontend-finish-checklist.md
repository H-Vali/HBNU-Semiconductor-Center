# Frontend Finish Checklist

HBNU Semiconductor Center 프론트엔드 마감 체크리스트입니다.

상태 표기:
- `[ ]` 대기
- `[~]` 진행중
- `[x]` 완료

## 1. 전체 페이지 반응형 QA

- [ ] 데스크톱 주요 해상도 확인
- [ ] 태블릿 폭 확인
- [ ] 모바일 폭 확인
- [ ] 히어로, 사이드바, 테이블, 캘린더, 모달 레이아웃 깨짐 확인
- [ ] 텍스트 겹침/잘림/버튼 overflow 확인

- [ ] 예약관리 UX 점검
- [ ] 사용자관리 UX 점검
- [ ] 권한관리 UX 점검
- [ ] 장비관리 UX 점검
- [ ] 소모품관리 UX 점검
- [ ] 공지사항관리 UX 점검
- [ ] 패널티관리 UX 점검
- [ ] 교육관리 UX 점검

- [ ] 저장 성공/실패 상태
- [ ] 삭제 확인/취소 상태
- [ ] Excel 업로드 성공/실패/형식 오류 상태
- [ ] 파일 첨부 성공/실패/형식 오류 상태
